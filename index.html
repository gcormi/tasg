Guide d'utilisation

Philosophie & Intégration ENT

1. Philosophie

Cet outil est un assistant pour structurer et créer du contenu rapidement.

Vous restez le seul maître de vos choix pédagogiques. Utilisez l'IA comme point de départ, puis adaptez.

2. Intégration Léia by Néo

L'application est optimisée pour l'ENT académique :

  • Exercices "Création Assistée" (PNG) Générez un fond d'exercice en image, importez-le dans Léia, et ajoutez les zones de réponse par-dessus.
  • Exercices Moodle (XML) Pour correction auto. Générez le XML, et importez-le dans la banque de questions d'un "Test" Léia.

3. Bonnes Pratiques

  • Soyez précis dans vos demandes (prompts).
  • Vérifiez toujours les contenus générés avant diffusion.

Fonctionnement API

Google Gemini (Gratuit)

Recommandé pour débuter. Obtenez une clé gratuite via Google AI Studio. Suffisant pour 90% des usages.

OpenRouter (Avancé)

Pour accéder à Claude 3.5, GPT-4o, etc. Nécessite un compte OpenRouter (coût à l'usage, très faible).

Le système détecte automatiquement le type de clé saisie.
