📖 Mode d'emploi

Cet outil est conçu pour vous faciliter la surveillance de l'épreuve :

  • 1. Préparez l'épreuve : Utilisez le menu déroulant pour sélectionner l'épreuve du jour. Toutes les informations s'ajusteront automatiquement.
  • 2. Ajustez les règles : Le bouton « 📋 Règles » permet d'afficher ou masquer certaines consignes selon l'épreuve.
  • 3. Donnez le Top Départ : Cliquez sur « ▶️ Maintenant » pour saisir l'heure exacte en un clic.
  • 4. Tiers-Temps : Cliquez sur « ⏱️ Afficher 1/3 Tiers-Temps » pour les horaires aménagés.
  • 5. N° du Candidat : Cliquez directement sur la case sur la copie virtuelle pour afficher « NE PAS REMPLIR ».
  • 6. Alertes sonores : Activez le toggle « 🔊 Sons » pour recevoir un bip à 30 min, 5 min et à la fin de l'épreuve.
  • 7. Masquer / Plein Écran : Cliquez sur « 👁️‍🗨️ Masquer » pour cacher le panneau surveillant, ou sur « 🔲 Plein Écran ».
  • 8. Raccourcis clavier : Échap pour fermer les fenêtres modales.
