Générateur d'activités pédagogiques

Configurez une série d'activités basées sur votre cours et générez un prompt pour votre IA.

À quoi servent ces 4 types d'activités ?
  • Identification : Permet de vérifier si l'étudiant peut reconnaître et nommer les concepts clés du cours (QCM, glisser-déposer sur un schéma, etc.).
  • Séquençage : Idéal pour les processus ou les chronologies. L'étudiant doit remettre des étapes dans le bon ordre.
  • Synthèse : Demande à l'étudiant de compléter un texte à trous résumant les idées principales, forçant la réappropriation des termes.
  • Logique inversée : Force une lecture critique. L'étudiant doit identifier l'affirmation incorrecte ou "l'intrus" dans un ensemble de faits.
