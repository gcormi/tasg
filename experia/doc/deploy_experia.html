Déploiement Experia

Installation automatique du proxy Cloudflare en 3 étapes — aucune connaissance technique requise.

1 Créer un compte Cloudflare gratuit

Si vous avez déjà un compte Cloudflare, passez directement à l'étape 2.
  1. Allez sur dash.cloudflare.com/sign-up
  2. Entrez votre adresse e-mail et créez un mot de passe
  3. Validez votre e-mail (lien envoyé par Cloudflare)
  4. Vous êtes dans le tableau de bord Cloudflare — continuez ci-dessous.

2 Créer un jeton API Cloudflare

Ce jeton permet à cette page de déployer le Worker en votre nom. Il ne sera jamais stocké.
  1. Dans le tableau de bord Cloudflare : cliquez sur votre icône de profil (en haut à droite) → Mon profil
  2. Menu de gauche : Jetons API
  3. Cliquez Créer un jeton
  4. Choisissez le modèle « Edit Cloudflare Workers »
  5. Cliquez Continuer vers le récapitulatif puis Créer un jeton
  6. Copiez le jeton affiché (il ne sera montré qu'une seule fois)

Format : une longue chaîne de lettres et chiffres
Visible dans Aperçu du compte → Identifiant du compte (colonne droite du tableau de bord).

3 Vos identifiants pédagogiques

Dans Nuage : cliquez sur votre avatar → Paramètres → Sécurité → Créer un nouveau mot de passe d'application
Obtenir une clé : albert.api.etalab.gouv.fr → se connecter avec ProConnect
Ce mot de passe vous sera demandé à chaque connexion dans prof.html. Retenez-le.
Utilisé dans l'URL finale. Ne mettez que des lettres minuscules et des tirets.
