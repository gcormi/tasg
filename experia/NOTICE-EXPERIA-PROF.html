1 Qu'est-ce qu'Experia ?

Experia est une application web éducative qui permet à un professeur de :

  • Fournir un corpus — texte libre ou fichier PDF/TXT (cours, programme, article…)
  • Générer automatiquement des activités — l'IA Albert crée les questions à partir de votre contenu
  • Partager une session — les élèves accèdent via un lien + code
  • Recevoir des résultats auto-corrigés — note globale calculée automatiquement, sans correction manuelle
Experia fonctionne avec l'IA souveraine Albert (Étalab / État français) et stocke les données sur votre espace Nuage apps.education.fr. Aucune donnée ne quitte les serveurs de l'Éducation Nationale.

2 Connexion à votre espace Nuage

Experia sauvegarde les sessions et résultats dans un dossier experia/ de votre Nuage personnel.

Étape 1 — Générer un mot de passe d'application

  1. Connectez-vous sur nuage.apps.education.fr
  2. Allez dans Paramètres → Sécurité → Mots de passe d'application
  3. Tapez un nom (ex: Experia) et cliquez Créer
  4. Copiez le mot de passe généré (il ne s'affiche qu'une seule fois !)

Étape 2 — Créer le dossier Experia sur Nuage

Dans l'interface web de votre Nuage, créez un dossier nommé exactement experia (en minuscules) à la racine de vos fichiers.

Étape 3 — Se connecter dans Experia

Serveur Nuage nuage03.apps.education.fr
Identifiant votre identifiant académique (ex: gcormi)
Mot de passe le mot de passe d'application généré à l'étape 1
Votre vrai serveur Nuage peut être différent (nuage01, nuage02, nuage03…). Vérifiez-le dans Fichiers → Paramètres → WebDAV sur votre Nuage.

3 Créer une session

Titre de la session

Donnez un titre clair que vos élèves verront (ex: "La vie de Mozart", "Cycle 4 — Les matériaux").

Le corpus

C'est le contenu à partir duquel l'IA génère toutes les questions. Deux options :

  • Texte libre — coller directement votre cours, vos notes, une description de programme
  • Fichier PDF ou TXT — importer un document, le texte est extrait automatiquement
Plus le corpus est précis et structuré, plus les questions générées seront pertinentes. Idéalement entre 200 et 3000 mots.

Configuration des activités

Pour chacune des 6 activités, choisissez le nombre de questions à générer :

Quiz QCMrecommandé : 8–15
Paires à relierrecommandé : 6–10
Mission rapiditérecommandé : 5–8
Compléter la phraserecommandé : 4–8
Détection d'intrusrecommandé : 4–6
Questions ouvertes IArecommandé : 2–4

Activités rejouables

Si cette option est cochée, les élèves peuvent recommencer une activité pour améliorer leur score. Décochez pour une évaluation en une seule tentative (comme un devoir).

Clé API Albert

Entrez votre clé API Albert (Étalab). Elle est utilisée pour générer les questions et évaluer les réponses ouvertes. Elle est stockée dans votre session sur votre Nuage personnel.

La génération prend 30 à 90 secondes selon le nombre d'activités et la taille du corpus. Une barre de progression vous indique l'avancement activité par activité.

4 Partager le lien aux élèves

Une fois la session générée, deux informations sont à communiquer à vos élèves :

Le lien d'accès

URL vers eleve.html avec la configuration encodée. À copier depuis le bouton "Copier".

Le code session

Format EXP-XXXX. L'élève le saisit manuellement pour vérification.

Comment partager ? Via votre ENT, Pronote, un message de classe, un QR code généré depuis le lien, ou simplement en projetant le code au tableau.

5 Consulter les résultats

Dès qu'un élève a soumis ses résultats, ils apparaissent dans l'onglet de la session.

Score global automatique

Moyenne des pourcentages de réussite de chaque activité. Calculé sans aucune action du professeur.

Détail par activité

Score et pourcentage pour chacune des 6 activités. Permet d'identifier les points faibles d'un élève.

Note et appréciation (optionnel)

Vous pouvez ajouter une note /20 et un commentaire. Ces informations sont sauvegardées sur votre Nuage.

6 Les 6 activités auto-corrigées

Quiz QCM — Questions à 4 choix. Correction immédiate avec explication. Correction : comparaison à la bonne réponse.
Paires à relier — Associer termes et définitions. Correction : vérification des associations.
Mission rapidité — QCM avec chrono 10s. Bonus de points pour la rapidité. Correction : QCM + bonus temps.
Compléter la phrase — Trouver le mot manquant dans une phrase. Tolérance aux fautes légères. Correction : distance de Levenshtein.
Détection d'intrus — Parmi 4 affirmations, repérer la fausse. Correction : comparaison à l'indice de la fausse affirmation.
Questions ouvertes IA — Réponse libre, notée par Albert sur 10 avec justification. Correction : évaluation par l'IA Albert via le corpus de référence.

7 Confidentialité & RGPD

Aucune donnée personnelle n'est collectée par l'application. Les données sont stockées uniquement sur votre espace Nuage personnel (apps.education.fr).
Les élèves s'identifient avec un identifiant libre (pas de nom, pas d'email, pas de connexion SSO). C'est vous qui choisissez la convention (numéro élève, prénom, pseudo…).
La clé API Albert et les identifiants Nuage sont transmis uniquement via le relais n8n de l'incubateur de l'Éducation Nationale. Ils ne transitent pas par des serveurs tiers.
