Pourquoi utiliser cet outil ?

Le collège marque une explosion de la charge cognitive : les élèves doivent simultanément décoder des textes complexes, organiser leurs idées et maîtriser de nouveaux outils. Pour un élève neuroatypique (Dys, TDAH), l'obstacle n'est souvent pas le concept lui-même, mais la forme sous laquelle il est présenté.

Cette plateforme pratique la remédiation cognitive : par l'usage du Meta-Prompting, nous automatisons l'application de protocoles ergonomiques et linguistiques éprouvés. L'objectif n'est jamais de simplifier le niveau d'exigence, mais de lever les barrières d'accès pour garantir que chaque élève accède à la totalité du savoir, conformément aux attendus du socle commun.

Fonctionnement de l'outil : Mode d'emploi

1

Chargement

Importez votre document original (DOCX, PDF, HTML). Assurez-vous que le texte intégral s'affiche correctement dans la zone de saisie pour une analyse exhaustive.

2

Paramétrage

Cochez les profils d'apprentissage ciblés. Les instructions d'adaptation varieront automatiquement selon le trouble et l'intensité d'accompagnement choisis.

3

Finalisation

Copiez le prompt généré et collez-le directement dans votre interface d'IA générative habituelle. L'IA appliquera alors les transformations nécessaires pour produire votre support adapté.

Conseil d'Expert :

"Le prompt généré intègre des directives de parallélisme strict pour éviter les résumés. Privilégiez l'utilisation d'un modèle d'IA performant en traitement de texte pour garantir que la structure et les nuances de votre cours soient parfaitement respectées."
