Déploiement — que faire quand tu modifies un fichier

⚡ Mettre à jour une page Experia (prof.html, eleve.html, guide…)

cp experia/prof.html deploy_site/experia/prof.html cp experia/eleve.html deploy_site/experia/eleve.html firebase deploy
Depuis le dossier frojet/

🧑‍🏫 Mettre à jour une page Mentoria (index.html, compagnion.html, guide…)

cp mentoria/index.html deploy_site/mentoria/index.html cp mentoria/compagnion.html deploy_site/mentoria/compagnion.html firebase deploy
Depuis le dossier frojet/

⚡ Mettre à jour le Worker Experia

dash.cloudflare.com → Workers & Pages → experia-proxy → Edit code → Ctrl+A → Coller → Deploy
Fichier source : experia/experia_worker_v2.js

🧑‍🏫 Mettre à jour le Worker Mentoria

dash.cloudflare.com → Workers & Pages → mentoria-proxy → Edit code → Ctrl+A → Coller → Deploy
Fichier source : mentoria/worker.js
