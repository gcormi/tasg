NovaPéda

L'assistant pédagogique universel & générateur d'agents conversationnels.

Accéder à l'outil

1. C'est quoi concrètement ?

NovaPéda agit comme un "Middleware" (une surcouche intelligente) entre vous et les Intelligences Artificielles génériques (ChatGPT, Mistral, Claude). Au lieu de devoir rédiger vous-même des prompts complexes ("Agis comme un expert, fais-moi un cours..."), NovaPéda vous propose des formulaires pédagogiques simples (cases à cocher, menus déroulants).

En arrière-plan, l'outil transforme vos clics en un "Super-Prompt" de plus de 50 lignes, hyper-structuré, optimisé pour obtenir le meilleur résultat possible de l'IA.

2. Fonctionnalités Détaillées

Didak'Bots (Chatbots Élèves)

Créez une page web de discussion sécurisée pour vos élèves.

  • Tuteur de méthode : Guide l'élève sans donner la réponse.
  • Personnage Historique : Incarne Napoléon, Marie Curie...
  • Débatteur Socratique : Challenge les arguments de l'élève.
Ingénierie de Cours

Générez toute la documentation enseignante.

  • Séquence complète : Déroulé séance par séance.
  • Différenciation : Adaptation de textes pour DYS / Allophones.
  • Évaluation : Création de sujets d'examen avec corrigé.

Comparatif : NovaPéda vs Prompts de Prof

Attention, ces deux outils semblent faire la même chose (fournir des prompts), mais leur usage diffère :

  • Utilisez NovaPéda si : Vous voulez du "Sur-Mesure". Vous remplissez un formulaire pour que l'IA génère un contenu spécifique à VOS besoins (votre niveau, votre chapitre précis).
  • Utilisez Prompts de Prof si : Vous voulez de la rapidité (Bibliothèque). Vous cherchez un modèle standard ("Prompt type pour corriger des copies") à copier-coller tel quel.

Modèles IA Supportés

NovaPéda ne "contient" pas l'IA, il s'y connecte ou vous donne le prompt pour elle.

Prompts de Prof

La grande bibliothèque de prompts prêts à l'emploi.

Accéder à la banque

1. C'est quoi concrètement ?

C'est un catalogue collaboratif hébergé sur la Forge. Contrairement à un générateur, il ne vous demande pas de paramètres. Il vous présente une liste de "recettes de cuisine" (prompts) qui ont été testées et validées par d'autres enseignants.

L'interface est minimaliste : vous cherchez un besoin (ex: "Créer un QCM"), vous cliquez, vous copiez le texte, et vous le collez dans votre IA.

Point de Vigilance : Redondance avec NovaPéda

Ne confondez pas les deux outils. "Prompts de Prof" est statique (une liste fixe), alors que "NovaPéda" est dynamique (il écrit le prompt pour vous).

Conseil d'Expert : Commencez par "Prompts de Prof" pour comprendre la structure d'un prompt, puis passez à "NovaPéda" pour industrialiser votre production.

DictIA

Générateur de textes sur-mesure pour la lecture et l'orthographe.

Accéder à l'outil

1. C'est quoi concrètement ?

DictIA résout un problème classique : trouver un texte intéressant qui respecte exactement les contraintes de votre classe à un instant T (ex: "Je veux du Passé Simple, mais pas de mots compliqués, et ça doit parler de Pirates").

C'est un outil de Différenciation Massive : en un clic, il vous donne non seulement le texte, mais aussi ses versions adaptées pour les élèves en difficulté.

2. Fonctionnalités Détaillées

🎛️ Paramétrage Fin (Les Entrées)

  • • Niveaux : Du CP à la 3ème (vocabulaire ajusté).
  • • Temps Verbaux : Cochez uniquement ce que vous avez étudié (ex: Présent + Futur). L'IA n'utilisera AUCUN autre temps.
  • • Mots Imposés : Collez votre liste de mots de la semaine. Ils seront intégrés naturellement dans l'histoire.
  • • Longueur : Courte (flash), Moyenne ou Longue (brevet).

✨ Les 3 Sorties Simultanées (Outputs)

Standard
Texte maître pour l'enseignant.
À Trous
Ciblage intelligent (verbes, pluriels...).
DYS / Adapté
Police adaptée, espacements, phrases simples.

4. Tutoriel & Workflow

Comment "Construire" sa dictée :
  1. Ouvrez l'interface. Sélectionnez le niveau (ex: CM1 - Période 3).
  2. Dans "Temps", cochez Imparfait. Décochez le reste.
  3. Dans "Mots à inclure", écrivez : château, douves, pont-levis, seigneur.
  4. Dans "Thème", mettez : Une attaque nocturne.
  5. Cliquez sur le bouton "Copier le prompt pour ChatGPT/Mistral" ou utilisez le lien direct si configuré.
  6. Collez ce prompt dans votre IA conversationnelle préférée.
  7. Copiez-collez le résultat (les 3 versions) dans votre traitement de texte habituel (Word/LibreOffice) pour la mise en page finale.

LMS & Interactivité Productivité

Caramel

Le constructeur automatique d'exercices H5P et Moodle.

Accéder à l'outil

1. C'est quoi concrètement ?

Caramel est un convertisseur intelligent. Il prend vos documents statiques (PDF de cours, listes de vocabulaire) et les transforme en fichiers d'activités (.h5p, .mbz) prêts à être envoyés sur votre plateforme d'apprentissage (ENT, Moodle).

Son point fort est l'utilisation de l'IA pour "comprendre" votre cours et inventer des questions pertinentes (QCM, Vrai/Faux) automatiquement.

2. Fonctionnalités Détaillées

Type d'Activité Ce que ça fait Format Export
Quiz (Questions Set) Série de QCM, Vrai/Faux générés depuis un texte. .h5p
Flashcards Cartes mémoires (Recto = Question, Verso = Réponse). .h5p
Mots Croisés Grille interactive générée depuis des définitions. .h5p
Jeu Millionnaire Jeu type TV avec paliers et jokers. .mbz
Banque Questions Import massif de questions pour tests notés. .xml

3. Matrice Technique (Inputs/Outputs)

Activité Entrée Sortie Intégration

4. Tutoriel : PDF vers Quiz H5P

1.

Sur Caramel, choisissez la tuile "Quiz (Question Set) H5P".

2.

Activez l'option "À partir d'un document". Glissez votre fichier PDF (ex: Cours sur la Seconde Guerre Mondiale).

3.

L'IA lit le document localement (votre navigateur travaille, rien ne part sur un serveur). Elle extrait les notions clés.

4.

Cliquez sur "Générer". Une liste de questions apparaît à droite. Vous pouvez les modifier, supprimer les mauvaises.

5.

Cliquez sur "Télécharger le .h5p".

6.

Ouvrez votre ENT/Moodle -> Banque de contenus -> Téléverser. Votre quiz est prêt et jouable.

Multimédia Sciences

Hub HAPI

La boîte à outils multimédia pour des exports ultra-légers (SCORM).

Accéder à l'outil

1. C'est quoi concrètement ?

Le Hub HAPI (Normandie) est une suite d'utilitaires pour créer des "grains" pédagogiques autonome. Son grand atout est le SCORM Léger : créer des exercices interactifs qui pèsent quelques kilo-octets (contrairement aux logiciels lourds du marché) et qui s'intègrent partout en remontant la note de l'élève.

2. Les 3 Outils Phares

Générateur SCORM

Créez des mini-jeux :

  • Tri (Catégorisation)
  • Chronologie
  • Image Interactive

Visionneuse 3D

Pour la Physique-Chimie :

  • Import fichiers .mol/.pdb
  • Recherche par nom ("Eau")
  • Export WebGL interactif

Fusion H5P

Organisation :

  • Combine plusieurs .h5p
  • Crée un livre numérique
  • Fluidifie le parcours

3. Matrice Technique

Module Entrée Sortie Destination
Générateur SCORM Images / Textes .zip (SCORM) Tout LMS (Noté)
Visionneuse 3D .mol / .pdb .glb / .h5p Visualisation WebGL
Générateur H5P Texte brut .h5p Import H5P

4. Tutorial : Créer un exercice de Tri (Catégorisation)

  1. Allez sur Hub HAPI -> Générateur SCORM -> Catégorisation.
  2. Définissez vos colonnes (ex: "Acide", "Basique", "Neutre").
  3. Ajoutez vos éléments à trier. Cela peut être du texte ("Jus de citron") ou des images importées.
  4. Paramétrez les feedbacks (bravo/échec).
  5. Cliquez sur Télécharger (ZIP).
  6. Dans Moodle/Éléa : Ajoutez une activité "Paquet SCORM", glissez le ZIP. L'exercice s'affiche et la note remonte automatiquement.

Esprit Critique & IA

Le Protocole A.U.D.I.T. : Analyser, Utiliser, Douter, Interroger, Tester.

Accéder aux ressources

1. C'est quoi concrètement ?

C'est une ressource pédagogique essentielle pour ne pas subir l'IA. Face à des élèves fascinés par la "magie" de ChatGPT, ce site propose des séquences prêtes à l'emploi pour adopter la posture du "Pilote de l'IA" plutôt que celle du passager passif.

L'objectif est d'activer le "Mode Columbo" : apprendre à traquer les incohérences, vérifier les sources et comprendre que l'IA ne dit pas la vérité, mais prédit le mot le plus probable.

2. Le Protocole A.U.D.I.T. Détaillé

A

Analyser

Comprendre comment ça marche (probabilités, jetons). Démystifier la "boîte noire".

U

Utiliser

Apprendre à prompter. Savoir assigner un rôle, donner un contexte et un format de sortie.

D

Douter

Le cœur du réacteur. Identifier les hallucinations (erreurs factuelles confiantes) et les biais.

I

Interroger

Questionner l'éthique. D'où viennent les données ? Quel impact environnemental ? Qui modère ?

T

Tester

Mettre l'IA à l'épreuve par l'expérimentation concrète en classe.

3. Matrice Pédagogique (Exemples)

Module / Scénario Activité Élève Compétence Travaillée
Détecter le style IA Comparer un texte humain et un texte machine sur un même sujet. Analyse Stylistique
L'IA peut-elle mentir ? Poser des questions pièges ou mathématiques complexes. Vérification des faits
Red Teaming Essayer de faire dire des bêtises ou contourner les règles de l'IA (en sécurité). Compréhension des limites

4. Exemple : Le "Mode Columbo"

"Dans la série, Columbo connaît le coupable dès le début. Son travail n'est pas de découvrir qui c'est, mais de construire la preuve."

Application en classe :
  • L'élève génère une réponse avec l'IA (ex: une démonstration mathématique ou une biographie).
  • Il ne doit pas "copier" la réponse, mais l'auditer.
  • Il doit surligner les erreurs, les imprécisions ou les "tics de langage" de l'IA.
  • Note finale : Elle n'est pas sur la production de l'IA, mais sur la qualité de la critique de l'élève.
