☁️ Nuage — apps.education.fr

Accédez à vos bots depuis n'importe quel PC. Collez le lien de votre dossier Nuage partagé.

▶ Comment obtenir ce lien ?

1. Allez sur nuage.apps.education.fr
2. Créez un dossier Mentoria
3. Clic droit → Partager → activez "Autoriser l'édition"
4. Copiez le lien de partage et collez-le ici. Vos données restent sur le cloud français.

💾 Clé USB / Dossier local
