Détails de l'erreur...

Comme redouté, le serveur de l'État (Nextcloud Nuage) rejette cette tentative de connexion. Même avec votre bon mot de passe, c'est ce qu'on appelle une erreur CORS.

Pour faire simple : le serveur refuse formellement de transmettre vos précieux fichiers à une application située sur un autre domaine (comme votre ordinateur ou la Forge). Il s'agit d'une très bonne sécurité pour le Ministère, mais cela empêche la création d'une application "frontend pure" comme les nôtres.

Accès au Nuage

Connectez-vous à votre espace personnel apps.education.fr via l'API WebDAV.

Il est recommandé de créer un "mot de passe d'application" dans les paramètres de sécurité de votre Nuage.
